  1. 01

    Open Terminal — yes, you

    Terminal is just a text box where your Mac listens. Press ⌘ space, type Terminal, hit return. A plain window appears with a blinking cursor. That's the whole app.

    Terminal ⌘ space
  2. 02

    Paste the one line

    Copy the command from the top of this page, paste it into Terminal, and press return. It checks your Mac and installs anything missing: Homebrew (an app installer), Node.js (runs your prototype), Claude Code (the part you talk to), and Superwhisper (dictation, so you can literally talk).

    First time, installs can take a few minutes — let it run.

    If it asks for a password: that's your normal Mac login password. Terminal won't show dots while you type it. Type it blind and press return — that's normal.
    Safe to re-run. Run the same line tomorrow and it skips everything that's already installed and jumps straight to making a new project.
  3. 03

    Answer two questions

    It asks what to call your prototype — pick anything, like checkout-redesign — and what kind to make:

    Recommended

    Next.js — press 1

    A real interactive web app. Dashboards, multi-step flows, clickable everything. Best for almost every idea.

    Plain HTML — press 2

    One page that opens instantly in your browser. Great for a quick landing page or a static mockup.

    Every project comes pre-loaded with realistic fake data — users, orders, revenue, KPIs — so your first screen looks real from minute one.

  4. 04

    Sign in — once, ever

    When Claude opens for the first time, it pops your browser to sign in with your Anthropic account. Click through, come back to Terminal, done. You will never be asked again on this Mac.

  5. 05

    Now just talk to it

    This is the whole trick: describe outcomes, not code. Say what you want like you'd brief a designer. Claude writes everything, tells you what it did in plain language, and asks when it needs a decision. Steal one of these to start:

    Tip: typing cc in any new Terminal window is a shortcut for claude.
    Hate typing? Setup also installed Superwhisper — open it once from Applications, then hold its hotkey and speak. Your words get typed wherever your cursor is, including into Claude.
  6. 06

    See it live while you talk

    Open a second Terminal tab (⌘ T) and run these two lines, then visit localhost:3000 in your browser. The page updates by itself every time Claude changes something — keep it side-by-side and watch your words become screens.

Things people worry about

Can I break something on my Mac?

Realistically, no. Everything lives in one folder — ~/Prototypes — and prototypes are throwaway by design: no databases, no accounts, nothing deployed. Worst case, you delete the folder and run the line again.

I closed Terminal. How do I get back to my prototype?

Run the same one-liner and type the same project name — it offers to reopen it. Or in Terminal: cd ~/Prototypes/your-project then cc. Every project also has a GETTING_STARTED.md file with these exact steps.

Claude did something I didn't want.

Just tell it. "Undo that" or "that's not what I meant — I wanted X" works. It's a conversation, not a form; you can redirect at any point.

Where does the fake data come from?

Every project ships with a mock-data file (users, orders, KPIs, revenue). Need something specific — menu items, reservations, whatever your idea needs? Ask Claude: "add mock data for X" and build with that.

Something's stuck or looks broken.

Press ctrl+C in Terminal to stop whatever's running, then run the one-liner again. It's built to be re-run — it never double-installs. Still stuck? Ask Claude itself; debugging its own mess is one of its best skills.
